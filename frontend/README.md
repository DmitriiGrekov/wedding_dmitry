# Wedding Invitation - React App

Приглашение на свадьбу Дмитрия и Екатерины, построенное на React.

## Компоненты

### 1. EnvelopeAnimation
Анимированный конверт, который открывается при клике и показывает приглашение.

### 2. MainSection
Главная секция с именами молодоженов, датой свадьбы и обратным отсчетом.

### 3. Countdown
Компонент обратного отсчета до события (недели, дни, часы, минуты, секунды).

### 4. EventInfo
Секция с подробной информацией о свадьбе, включает:
- Приветствие
- Дату и время
- Расписание
- Форму подтверждения
- QR-код для Telegram

### 5. Schedule
Расписание свадебного дня с картой места проведения.

### 6. GuestForm
Форма для подтверждения присутствия (до 4 гостей).

### 7. TelegramQR
QR-код для присоединения к группе в Telegram.

## Установка и запуск

### Установка зависимостей
```bash
npm install
```

### Запуск в режиме разработки
```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:3000

### Сборка для продакшена
```bash
npm run build
```

Собранные файлы будут находиться в папке `dist/`

### Предпросмотр продакшен версии
```bash
npm run preview
```

## Технологии

- **React 18** - библиотека для создания пользовательского интерфейса
- **Vite** - сборщик и dev-сервер
- **qr-code-styling** - генерация стилизованных QR-кодов
- **CSS3** - стили с анимациями и адаптивным дизайном

## Настройка

### Изменение даты свадьбы
В файле `src/App.jsx` измените дату:
```javascript
const eventDate = new Date('2026-07-18T15:00:00+03:00');
```

### Изменение ссылки на Telegram
В файле `src/components/TelegramQR.jsx` или при использовании компонента:
```jsx
<TelegramQR telegramLink="https://t.me/ваша_ссылка" />
```
