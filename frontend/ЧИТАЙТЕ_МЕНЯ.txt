                    🆘 НУЖНА ПОМОЩЬ?
════════════════════════════════════════════════════════════════════════════

1. Приложение не запускается?
   rm -rf node_modules package-lock.json
   npm install

2. Ошибка при установке?
   Проверьте версию Node.js: node -v (нужна >= 16)

3. Изображения не отображаются?
   Проверьте, что файлы в папке public/

4. Порт 3000 занят?
   Измените в vite.config.js на другой порт
