# Документация компонентов

## Структура компонентов

```
App.jsx
├── EnvelopeAnimation (анимация при загрузке)
└── MainContent (показывается после анимации)
    ├── MainSection
    │   ├── Countdown (обратный отсчет)
    │   └── ScrollButton
    └── EventInfo
        ├── Schedule (расписание + карта)
        ├── GuestForm (форма гостей)
        └── TelegramQR (QR-код)
```

## Подробное описание компонентов

### 1. App.jsx
**Главный компонент приложения**

**Props:** Нет

**State:**
- `showContent` - показывать ли основной контент (после анимации конверта)

**Функции:**
- `handleAnimationComplete()` - вызывается после завершения анимации конверта

**Константы:**
- `eventDate` - дата события свадьбы

---

### 2. EnvelopeAnimation.jsx
**Анимированный конверт с приглашением**

**Props:**
- `onAnimationComplete` (function) - callback после завершения анимации

**State:**
- `animationStarted` (boolean) - флаг начала анимации

**Особенности:**
- Анимация открытия конверта
- Печать исчезает
- Клапан открывается
- Письмо выдвигается
- Через 4 секунды вызывается callback

**Стили:** `EnvelopeAnimation.css`

---

### 3. MainSection.jsx
**Главная секция с именами и обратным отсчетом**

**Props:**
- `eventDate` (Date) - дата события

**Компоненты:**
- Левое и правое фоновое изображение
- Центральный блок с содержимым
- Компонент Countdown
- Кнопка прокрутки вниз

**Функции:**
- `scrollToEventInfo()` - плавная прокрутка к секции EventInfo

**Стили:** `MainSection.css`

---

### 4. Countdown.jsx
**Таймер обратного отсчета**

**Props:**
- `eventDate` (Date) - дата события

**State:**
- `timeLeft` (object) - объект с полями: weeks, days, hours, minutes, seconds

**Особенности:**
- Обновляется каждую секунду
- Показывает недели, дни, часы, минуты, секунды
- После наступления события показывает сообщение

**Стили:** `Countdown.css`

---

### 5. EventInfo.jsx
**Секция с информацией о событии**

**Props:** Нет

**Компоненты:**
- Приветствие
- Информация о дате
- Schedule (расписание)
- GuestForm (форма подтверждения)
- TelegramQR (QR-код)

**Стили:** `EventInfo.css`

---

### 6. Schedule.jsx
**Расписание свадебного дня**

**Props:** Нет

**Структура:**
- Список событий с временем
- Карта Яндекс с местом проведения (iframe)

**Стили:** `Schedule.css`

---

### 7. GuestForm.jsx
**Форма для подтверждения гостей**

**Props:** Нет

**State:**
- `guests` (array) - массив объектов с полями lastName и firstName

**Константы:**
- `MAX_GUESTS = 4` - максимальное количество гостей

**Функции:**
- `handleAddGuest()` - добавление нового гостя
- `handleRemoveGuest(index)` - удаление гостя
- `handleInputChange(index, field, value)` - изменение данных гостя
- `handleSubmit(e)` - отправка формы

**Особенности:**
- Минимум 1 гость, максимум 4
- Поля: Фамилия, Имя
- Кнопка удаления появляется при добавлении второго гостя
- После отправки форма сбрасывается

**Стили:** `GuestForm.css`

---

### 8. TelegramQR.jsx
**QR-код для Telegram группы**

**Props:**
- `telegramLink` (string, default: 'https://t.me/ВАША_ССЫЛКА_НА_ГРУППУ')

**State:** Нет

**Refs:**
- `qrRef` - ссылка на DOM элемент для QR-кода
- `qrCodeRef` - ссылка на экземпляр QRCodeStyling

**Библиотека:** qr-code-styling

**Особенности:**
- Стилизованный QR-код с округлыми углами
- Прозрачный фон
- Цвет: #ecfbff
- Размер: 200x200px
- Кнопка-ссылка на группу

**Стили:** `TelegramQR.css`

---

## Адаптивность

Все компоненты адаптивны и имеют брейкпоинты:
- **Desktop:** > 900px
- **Tablet:** 600px - 900px
- **Mobile:** < 600px

## Шрифты

- **Great Vibes** - имена молодоженов (cursive)
- **Playfair Display** - основной текст (serif)
- **Rubik Mono One** - заголовки секций (monospace)
- **Cormorant Garamond** - текст в конверте (serif)

## Цветовая палитра

- **Золотой акцент:** #d4af37
- **Желтый заголовок:** #ffe9a7
- **Синяя кнопка:** #4e6fa4
- **Белый текст:** #ecfbff
- **Темный текст:** #2c3e50

## Анимации

1. **Конверт:**
   - Печать исчезает (fade-out)
   - Клапан открывается (rotateX)
   - Письмо выдвигается (translateY)
   - Конверт опускается (translateY)

2. **Кнопка прокрутки:**
   - Bounce анимация

3. **Подсказка "Нажмите":**
   - Float анимация (плавное движение вверх-вниз)

## Зависимости

- **React 18** - библиотека UI
- **qr-code-styling** - генерация QR-кодов
- **Vite** - сборщик и dev-сервер
